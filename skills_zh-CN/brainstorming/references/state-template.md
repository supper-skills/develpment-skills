# 状态文件模板

本文档提供头脑风暴会话状态文件的模板结构，用于解决多轮对话中的注意力丢失问题。

---

## 状态文件路径

```
design/specs/YYYY-MM-DD-<topic>-state.md
```

---

## 完整模板

```markdown
# 头脑风暴会话状态

## 元信息

- **会话主题**: [需求简述]
- **创建时间**: YYYY-MM-DD HH:mm
- **最后更新**: YYYY-MM-DD HH:mm

## 当前阶段

阶段 X/8: [阶段名称]

## 已确认的规格

<!-- 用户已明确确认的内容 -->

### 目标定位

- ...

### 约束条件

- ...

### 成功标准

- ...

## 待决策问题

<!-- 需要向用户询问的问题队列 -->

1. [ ] 问题1
2. [ ] 问题2

## 已做出决策

<!-- 记录所有决策及其理由 -->

- 决策: ... | 理由: ...

## 下一步行动

[具体行动描述]
```

---

## 使用说明

### 每轮对话开始时

1. 读取状态文件
2. 根据状态文件恢复上下文
3. 声明当前阶段

### 每次获得用户确认后

1. 更新状态文件
2. 检查是否可进入下一阶段
